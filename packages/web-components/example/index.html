Standards-based custom elements with tree-shakable renderer support

üå≥ Tree-Shakable: Import only the renderers you need. Each component bundles only its specific renderer, keeping your bundle size minimal.

Vanilla CSS Renderer

Pure CSS styling, no framework dependencies.

<!-- Import the component -->
<script type="module">
  import '@touchspin/web-components/vanilla';
</script>

<!-- Use it in your HTML -->
<touchspin-vanilla min="0" max="100" value="50"></touchspin-vanilla>

Bootstrap 5 Renderer

Fully integrated with Bootstrap 5 components and utilities.

<!-- Import Bootstrap 5 CSS -->
<link rel="stylesheet" href="bootstrap.min.css">

<!-- Import the component -->
<script type="module">
  import '@touchspin/web-components/bootstrap5';
</script>

<!-- Use it in your HTML -->
<touchspin-bootstrap5
  min="0"
  max="100"
  value="25"
  prefix="$"
  postfix=" USD">
</touchspin-bootstrap5>

üìù Programmatic API

All components expose the same API for programmatic control.

const spinner = document.querySelector('#api-example');

// Methods
spinner.upOnce();          // Increment once
spinner.downOnce();        // Decrement once
spinner.setValue(75);      // Set value
const value = spinner.value; // Get value

// Properties
spinner.min = 0;
spinner.max = 100;
spinner.step = 5;
spinner.disabled = true;
spinner.readonly = true;

// Events (dispatched as CustomEvents)
spinner.addEventListener('touchspin.on.min', (e) => {
  console.log('Minimum reached:', e.detail);
});
spinner.addEventListener('touchspin.on.max', (e) => {
  console.log('Maximum reached:', e.detail);
});
