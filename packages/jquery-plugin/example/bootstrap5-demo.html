Bootstrap 5 Features

This demo showcases TouchSpin working with Bootstrap 5.3.0, featuring:

  • Simplified input group structure (no wrapper divs)
  • Bootstrap Icons integration
  • CSS custom properties (CSS variables)
  • Enhanced color system and utilities
  • Improved accessibility features

Basic Usage

Code:
$("input[name='demo1']").TouchSpin();
Code:
$("input[name='demo2']").TouchSpin({
  min: 0,
  max: 100,
  step: 5
});

Currency Formatting

Code (with callbacks):
const before = (raw) => (raw ?? '').replace(/[^\d\.\-]/g, '');
const after = (numStr) => {
  const n = parseFloat(numStr);
  if (!isFinite(n)) return '';
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency: 'USD',
    minimumFractionDigits: 2,
    maximumFractionDigits: 2
  }).format(n);
};

$("input[name='demo3']").TouchSpin({
  min: 0,
  max: 100000,
  step: 0.01,
  decimals: 2,
  callback_before_calculation: before,
  callback_after_calculation: after
});

Vertical Buttons

Code:
$("input[name='demo4']").TouchSpin({
  verticalbuttons: true,
  verticalup: '<i class="bi bi-chevron-up"></i>',
  verticaldown: '<i class="bi bi-chevron-down"></i>'
});
