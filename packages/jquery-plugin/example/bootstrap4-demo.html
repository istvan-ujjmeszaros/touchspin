Bootstrap 4 Features

This demo showcases TouchSpin working with Bootstrap 4.6.2, featuring:

  • Bootstrap 4's improved input group structure
  • Font Awesome icons integration
  • Enhanced spacing and typography utilities
  • Flexbox-based layout system

Basic Usage

Code:
$("input[name='demo1']").TouchSpin();
Code:
$("input[name='demo2']").TouchSpin({
  min: 0,
  max: 100,
  step: 5
});

Currency Formatting

Code (with callbacks):
const before = (raw) => (raw ?? '').replace(/[^\d\.\-]/g, '');
const after = (numStr) => {
  const n = parseFloat(numStr);
  if (!isFinite(n)) return '';
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency: 'USD',
    minimumFractionDigits: 2,
    maximumFractionDigits: 2
  }).format(n);
};

$("input[name='demo3']").TouchSpin({
  min: 0,
  max: 100000,
  step: 0.01,
  decimals: 2,
  callback_before_calculation: before,
  callback_after_calculation: after
});

Vertical Buttons

Code:
$("input[name='demo4']").TouchSpin({
  verticalbuttons: true,
  verticalup: '<i class="fas fa-chevron-up"></i>',
  verticaldown: '<i class="fas fa-chevron-down"></i>'
});

Input Groups

Code:
$("input[name='demo7']").TouchSpin({
  prefix: 'Distance:',
  postfix: 'km'
});

High Precision

Code:

  min: 0,
  max: 10,
