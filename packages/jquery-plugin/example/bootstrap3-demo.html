Bootstrap 3 Features

This demo showcases TouchSpin working with Bootstrap 3.4.1, featuring:

  • Classic Bootstrap 3 input group structure
  • Glyphicons for button icons
  • Traditional Bootstrap 3 color scheme
  • Full backwards compatibility with legacy projects

Basic Usage

Code:

$("input[name='demo1']").TouchSpin();

Code:

$("input[name='demo2']").TouchSpin({
  min: 0,
  max: 100,
  step: 5
});

Currency Formatting

Code (with callbacks):

const before = (raw) => (raw ?? '').replace(/[^\d\.\-]/g, '');
const after = (numStr) => {
  const n = parseFloat(numStr);
  if (!isFinite(n)) return '';
  return '$' + n.toLocaleString('en-US', {
    minimumFractionDigits: 2,
    maximumFractionDigits: 2
  });
};

$("input[name='demo3']").TouchSpin({
  min: 0,
  max: 100000,
  step: 0.01,
  decimals: 2,
  callback_before_calculation: before,
  callback_after_calculation: after
});

Vertical Buttons

Code:

$("input[name='demo4']").TouchSpin({
  verticalbuttons: true,
  verticalup: '<span class="glyphicon glyphicon-chevron-up"></span>',
  verticaldown: '<span class="glyphicon glyphicon-chevron-down"></span>'
});

Input Sizes

<input class="form-control input-sm" ...>

Input Groups

Code:

$("input[name='demo7']").TouchSpin({
