Bootstrap 3 Features

This demo showcases Bootstrap TouchSpin working with Bootstrap 3.4.1, featuring:

  • input-group-btn and input-group-addon classes
  • Bootstrap 3 button styles and sizing classes
  • Classic Bootstrap 3 form controls and layouts
  • Automatic Bootstrap version detection

Basic Examples

Code:

$("input[name='demo1']").TouchSpin();

Code:

$("input[name='demo2']").TouchSpin({
  prefix: '$',
  postfix: '.00',
  min: 0,
  max: 1000
});

Code:

$("input[name='demo3']").TouchSpin({
  min: 0,
  max: 10,
  step: 0.01,
  decimals: 2
});

Sizing Examples

Code:

<input class="form-control input-sm" ...>
$("input[name='demo4']").TouchSpin();

Code:

<input class="form-control input-lg" ...>
$("input[name='demo5']").TouchSpin();

Vertical Buttons

Code:

$("input[name='demo6']").TouchSpin({
  verticalbuttons: true,
  verticalup: '<i class="fa fa-plus"></i>',
  verticaldown: '<i class="fa fa-minus"></i>'
});
