Basic Examples

Simple increment/decrement with Tailwind styling

Raw HTML Input

<input type="number" id="basic-spinner" value="50" class="w-full">

JavaScript Initialization

$('#basic-spinner').TouchSpin({
  min: 0,
  max: 100,
  step: 5
});

Form Integration

TouchSpin integrates seamlessly with Tailwind form layouts - perfect for e-commerce and order forms.

Raw HTML Form

<form class="max-w-md space-y-6">
  <div>
    <label for="quantity" class="demo-label">Quantity</label>
    <input type="number" id="quantity" value="1" class="w-full">
  </div>
  
  <div>
    <label for="price" class="demo-label">Price</label>
    <input type="number" id="price" value="99.99" class="w-full">
  </div>
  
  <div class="pt-4">
    <button type="button" class="w-full bg-blue-600 hover:bg-blue-700 text-white font-medium py-2 px-4 rounded-lg">
      Add to Cart
    </button>
  </div>
</form>

JavaScript Initialization

// Quantity selector
$('#quantity').TouchSpin({
  min: 1,
  max: 999,
  step: 1
});

// Price with currency formatting
$('#price').TouchSpin({
  min: 0,
  max: 9999.99,
  step: 0.01,
  decimals: 2,
  prefix: '$'
});
