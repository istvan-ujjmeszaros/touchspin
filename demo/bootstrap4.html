Bootstrap 4 Features

This demo showcases Bootstrap TouchSpin working with Bootstrap 4.6.2, featuring:

  • input-group-prepend and input-group-append wrapper classes
  • input-group-text for addons and custom content
  • Bootstrap 4 button styles and modern form controls
  • Flexbox-based layout system
  • Enhanced spacing and typography

Basic Examples

Code:
$("input[name='demo1']").TouchSpin();
Code:
$("input[name='demo2']").TouchSpin({
  prefix: '$',
  postfix: '.00',
  min: 0,
  max: 1000
});
Code:
$("input[name='demo3']").TouchSpin({
  min: 0,
  max: 10,
  step: 0.001,
  decimals: 3
});
