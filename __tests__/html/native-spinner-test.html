TouchSpin Native Number Input Tests

This page tests how TouchSpin interacts with browser native number input controls.

Test 1: Number Input WITH Native Attributes

Input has native min="5", max="15", step="2" attributes.
TouchSpin configured with min: 3, max: 20, step: 1 (different values).
Question: Which settings will browser native spinners respect?
TouchSpin Configuration: 
$('#native-with-attrs').TouchSpin({
    min: 3,        // Conflicts with native min="5"
    max: 20,       // Conflicts with native max="15"  
    step: 1        // Conflicts with native step="2"
});

Test 2: Number Input WITHOUT Native Attributes

Input has NO native min/max/step attributes.
TouchSpin configured with min: 3, max: 20, step: 1.
Question: Will browser native spinners respect TouchSpin-only settings?
TouchSpin Configuration: 
$('#native-without-attrs').TouchSpin({
    min: 3,
    max: 20,
    step: 1
});

Test 3: Text Input (Control Group)

Input type="text" - no browser native spinners available.

Purpose: Control group to verify TouchSpin-only behavior.

Test 4: Decimal Number Input WITH Native Attributes

Input has native min="0.5", max="10.5", step="0.25" attributes.
TouchSpin configured with min: 0, max: 15, step: 0.5, decimals: 2 (different values).
Question: How do decimal native attributes interact with TouchSpin?
TouchSpin Configuration: 
$('#decimal-native-with-attrs').TouchSpin({
    min: 0,        // Conflicts with native min="0.5"
    max: 15,       // Conflicts with native max="10.5"
    step: 0.5,     // Conflicts with native step="0.25"
    decimals: 2
});
